# JoeCamelBot

JoeCamelBot — это Телеграм-бот, предназначенный для отслеживания и оценки активности пользователей в групповых чатах. Бот использует Google Gemini AI для анализа сообщений, начисления баллов и ведения рейтинга на основе системы "База против Анти".

## Основные компоненты

- main.py: Основная логика бота, обработчики событий (aiogram) и планировщик задач (apscheduler).
- database.py: Асинхронное управление базой данных SQLite (aiosqlite) для хранения статистики и истории.
- ai_logic.py: Интеграция с Google Gemini API для оценки сообщений и формирования ответов от лица персонажа.
- Dockerfile и docker-compose.yml: Контейнеризация и оркестрация для развертывания.

## Функциональные возможности

- AI-оценка: Автоматический анализ сообщений и начисление баллов (Мини, Средняя, Большая, Экстра, Мега).
- Персонаж: Ответы в стиле Джо Кэмела — циничного философа из рекламы 90-х, использующего аллюзии на экзистенциальную литературу.
- Диспуты: Возможность оспорить вердикт бота через систему социального голосования участников чата.
- Ежедневные штрафы: Автоматическое списание баллов за отсутствие активности для поддержания дисциплины.
- Аудиты: Периодические саммари последних событий в чате и случайные комментарии от бота.

## Требования

- Python 3.11 или выше.
- Docker и Docker Compose (рекомендуется).
- Токен Телеграм-бота (от @BotFather).
- Ключ Google Gemini API.

## Установка и запуск

1. Клонируйте репозиторий:
   git clone https://github.com/divohub/JoeCamelBot.git

2. Создайте файл .env в корневой директории:
   TELEGRAM_BOT_TOKEN=ваш_токен
   GEMINI_API_KEY=ваш_ключ
   ADMIN_CHAT_ID=id_администратора
   DB_PATH=data/shnyaga.db

3. Запустите приложение через Docker:
   docker-compose up -d --build

## Список команд

- /start: Инициализация профиля пользователя и краткое введение.
- /help: Просмотр правил и доступных команд.
- /top: Список лидеров чата по баллам силы.
- /stats: Просмотр личной статистики и истории деяний с пагинацией.
- /setchat: Настройка текущего чата как основной арены для отчетов.
